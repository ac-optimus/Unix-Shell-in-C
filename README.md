# Overview
Unix shell implementaiton in C. Shell uses mixture of commands that are implemented using c system calls and inbuild binaries that are already present in Unix.

## Following commands are implemented in c.
- [x] ls
- [x] grep
- [x] cat
- [x] mv
- [x] cp
- [x] cd
- [x] pwd
- [x] rm
- [x] Chmod
- [x] Mkdir
